## 🎨 项目特色

### 技术栈
- **前端**: Vue 3 + TypeScript + Element Plus + Vite
- **后端**: Flask + SQLAlchemy + MySQL
- **设计**: 翠绿色赛博朋克风格 + 粒子背景 + 交互特效

### 核心功能
- 用户注册登录
- 图书浏览和搜索
- 购物车管理
- 订单管理
- 后台管理系统

### 视觉特色
- 🎨 统一的翠绿色主题 (#10b981, #34d399, #6ee7b7, #059669)
- ✨ 动态粒子背景 (60fps流畅动画)
- 💫 鼠标交互效果 (粒子排斥、网格连线)
- 🖤 纯黑底色 (#000000)
- 🎯 瀑布流网格布局 (Pinterest风格)
